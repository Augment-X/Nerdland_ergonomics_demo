# Nerdland_ergonomics_demo

## NL
Deze repository bevat het programma voor automatische ergonomische beoordeling voorgesteld op het 2026 Nerdland festival. Volg deze stappen om het te installeren en gebruiken:

1. Installeer Python: ga naar https://www.python.org/downloads/ en klik op "Download Python install manager". Dubbelklik op het bestand om Python te installeren.
2. Download de bestanden uit deze repository: klik rechtsboven op de groene knop "Code" en vervolgens "Download ZIP". Unzip het bestand.
3. Installeer de benodigde Python packages: navigeer naar de map met "Nerdland_ergonomics_demo.py" in de verkenner en typ in de adresbalk "cmd" en druk op enter (of rechtsklik in de map en klik op "open in terminal"). Typ vervolgens "pip install -r requirements.txt" en druk op enter. De packages zouden vanzelf geïnstalleerd moeten worden.
4. Start het programma: Typ in dezelfde cmd terminal "python Nerdland_ergonomics_demo.py". Je zal een lijst met camera's te zien krijgen, waaruit je er 1 moet selecteren. Typ het overeenstemmende getal in (0-5) en druk op enter. Het programma zou nu vanzelf moeten opstarten. Druk op esc om het te sluiten. Als de camera niet de gewenste is, probeer het opnieuw met een ander getal.

## EN
This repository contains the program for the automatic ergonomic assessment presented at the 2026 Nerdland festival. Follow these steps to install and use it:

1. Install Python: go to python.org downloads page and click “Download Python install manager”. Double-click the downloaded file to install Python.
2. Download the files from this repository: click the green “Code” button in the top right corner and then select “Download ZIP”. Extract the ZIP file.
3. Install the required Python packages: navigate to the folder containing Nerdland_ergonomics_demo.py in File Explorer and type cmd in the address bar, then press Enter (or right-click inside the folder and select “Open in Terminal”). Then type "pip install -r requirements.txt" and press Enter. The required packages should install automatically.
4. Start the program: in the same terminal window, type "python Nerdland_ergonomics_demo.py" You will see a list of available cameras from which you must choose one. Enter the corresponding number (0–5) and press Enter. The program should now start automatically. Press Esc to close it. If the selected camera is not the desired one, restart the program and try a different number.
